# Air Quality Index (AQI) Analysis - Pakistan

## 📊 Project Overview
This project analyzes air quality trends across different cities in Pakistan using data visualization and analytics.

##  Tools Used
- Python (Pandas, NumPy)
- Power BI / Excel
- Data Cleaning & Visualization

## 📈 Key Insights
- Identified most polluted cities
- Analyzed AQI trends over time
- Compared regional air quality patterns

##  Features
- Interactive dashboard
- AQI trend analysis
- Geographic visualization
